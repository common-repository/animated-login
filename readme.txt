Replace the WP logo on the login page with your OWN image, then animate it!

== Description ==
Animated Login allows the user to swap out the WP logo on the login page with a custom image or logo, link it to the home page, and pair it with a fun CSS3 animation! 

= Features =

*	Compatible with IE9+, Chrome10+, Safari, FireFox
*	Over 50 animations to choose from.
*	Option to link your logo to the homepage.
*	Animation preview is available on the settings page.
*	Translation ready.

== Installation ==
1. Simply download the plugin and install.
2. Choose an image from the Media Library. Copy the full url and paste it into the text box. 
3. Set the animation delay.
4. Choose one of the super cool animations from the drop-down.
5. Save!

= Manual Installation =
Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
